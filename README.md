# Сортировка цветных шаров 

## Запуск проекта
### Инструкция 

1. Клонировать репозиторий 
```sh
git clone git@github.com:V1taly5/sortable.git
cd sortable
```

2. Запустить программу 
```sh
go run main.go
```

3. Пример ввода
```sh
2
1 0
0 1
```

4. Пример вывода 
```sh
yes
```

### Запуск тестов
```sh
go test -v
```
